API

Readsr offers an API for retrieving information about Series and Readings. Currently, GET requests are supported, and the API is read-only.

  • The Series endpoint is api/v1/series/<series id>/.
  • The Reading endpoint is api/v1/reading/<reading id>/.

Formats can be requested through the format parameter. Valid formats are:

  • json
  • xml
